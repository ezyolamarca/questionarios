Exercícios de Conversão de Bases Numéricas

Nome do aluno:

Nome deve ter ao menos 3 caracteres.

Turma:

Turma deve ter ao menos 3 caracteres.

Parte 1: Questões Objetivas

1. Converta o número decimal 25 para binário.





2. Qual é o valor hexadecimal correspondente ao número binário 11011110?





3. Converta o número binário 100110 para decimal.





4. O número decimal 64 equivale a qual valor em hexadecimal?





5. Qual é o valor em octal do número binário 101101?





6. Em uma tabela de endereçamento IP, um campo apresenta o valor 0xFF. Qual o valor decimal equivalente?





7. O valor decimal 127 corresponde a qual número binário?





8. Converta o número hexadecimal 1A para decimal.





9. Qual é a representação em hexadecimal do número binário 11110000?





10. Um código ASCII usa a notação hexadecimal 41 para representar uma letra. Qual é a letra?
